# CRUD Web Application

A basic CRUD (Create, Read, Update, Delete) web application built using core web technologies — HTML, CSS, and JavaScript. This project demonstrates dynamic client-side data handling and DOM manipulation without using any external libraries or frameworks.

---

## 🚀 Features

- Add new user data
- Display stored records dynamically
- Edit existing entries
- Delete records
- Client-side data management
- Responsive and clean UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 💡 Key Concepts Practiced

- DOM manipulation
- Event handling
- Form validation
- Dynamic content rendering
- Data management on the client side
